Técnico de telecomunicaciones

Por lo que veo, quieres saber mas sobre mi

Mi experencia con Telecomunicaciones abarca tanto Telefonia como Routers Cisco.

Routers y Redes.

Para la parte de routers, tengo experencia con las siguientes tareas:

  1. Revisión de enrutamiento, tanto Dinámico como Estático, comprobando que las rutas funcionan correctamente.
  2. Revisión de los puertos y sus direcciones IP.
  3. Revisión de conectividad correcta a la WAN y con los equipos locales mediante pruebas de Ping.
  4. Revisión de los túneles, que funcionen correctamente y reiniciarlos si hay algún inconveniente.
  5. Configuración completa del dispositivo en caso de sustitución.
  6. Instalación de certificados de seguridad.

Telefonia.

Para la parte de telefonia, tengo experencia con las siguientes tareas:

  1. Revision de equipos individuales de telefonia, como Centralitas.
  2. Revisión de los puertos y estado de los enlaces.
  3. Cambios en las lineas de cliente y configuraciones.
  4. Gestion de revision por parte de tecnicos de campo en caso de averia y posterior revision para confirmar reparacion.

Mi Bootcamp

Por lo que veo, quieres saber mas sobre mi

A finales de 2023 decidi reorientar mi carrera laboral con un Bootcamp en Data Science, donde adquiri muchas habilidades en Python mediante la realización de gran cantidad de proyectos.

My experencia en The Bridge.

Vi el Bootcamp como un desafio que superar, cada día estaba el objetivo de aprender algo nuevo y aplicarlo a ejercicios, para así, el día siguiente dar nueva materia, de esta forma, en tan solo 4 meses aprendi desde lo mas basico de Python hasta técnicas avanzas de Data Science.

Se culmino el curso con un trabajo real para la empresa Several Energy junto a distintos grupos como el de Cyberseguridad o UX/UI, donde pasamos su modelo de consultas de Excel a una base de datos en Python, automatizando varios procesos y mejorando notablemente la velocidad con la que podian dar consultas a cliente.

En general, fue una etapa realmente divertida donde aprendi muchisimo en poco tiempo y pude relacionarme con mucha gente, sin duda una experencia que recomendaria a aquellos que quieran aprender programación.

Te dejo aqui mi primer proyecto, un un hundir la flota para jugar, ¿porque no echas una partida y lo pruebas?

¿Que hice en el bootcamp?

  1. Programacion basica en Python, desde variables hasta trabajar con objetos.
  2. Manejo de las diferentes librerias basicas y especificas para Data Science y Machine learning.
  3. Exposición de trabajos para toda la clase con el objetivo de aprender a exponer el trabajo realizado.
  4. SQL para poder trabajar con bases de datos con los diferentes programas relacionados (mySQL, PostgreSQL, etc).
  5. Manejo de AWS para crear una base de datos y poder trabajar con técnologias en la nube.
  6. EDAs de eleccion propia donde tuve que buscar datos por mi cuenta y analizarlos para exponer delante de clase.
  7. Competiciones de Kaggle para hacer el mejor modelo predictivo.
  8. Proyectos con otros grupos para aprender a trabajar en sincronia con diferentes departamentos.

Mi Experencia como Analista en Argoz Consultants

Mi experencia como analista de datos proviene de la empresa Argoz consultants.

¿Que es Argoz Consultants?

Argoz Consultants es un grupo de analistas de datos y de expertos en medios de pago, enfocados asesorar a sus clientes para darles la mayor rentabilidad y crear soluciones de software para redefinir los estandares de la industria.

Trabajan con clientes de todo el mundo, dando sus servicios de consultoria.

Mi experencia como Analista.

Mi experencia en Argoz fue bastante buena, pude aprender mucho de Excel y Power Query para analisis de datos bancarios mediante creacion de tablas dinamicas, graficos y limpieza de datos para asegurar que estaban correctos.

Otra parte de mi trabajo consistio en la creacion de los procesos de ETL para SQLServer de tipo SSIS (SQL Server Integration Services) para almacenar los datos en las bases de SQL server y proceder a la categorización de datos desde Microsoft Access.

Por otro lado, me encarge de ayudar con modelos de Machine Learning desde Azure Databricks, dando soporte a un Técnico de Reino Unido para la creación de un modelo de Redes Neuronales.
